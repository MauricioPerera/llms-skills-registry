# mauricioperera.github.io

The reference multi-project origin of the ecosystem, and the best first stop:
one `llms.txt` serves `site_facts`, the standard's own skills
(`validate_skill_line`, `search_knowledge`, `list_concepts`) and the KDD
methodology bundle namespaced as `kdd__search_knowledge` /
`kdd__list_concepts` — each scope bound to its own hash-verified BM25 memory
(Executable Skills v0.5 §2.5).

## Consume

```
npx -y @rckflr/mcpwasm https://mauricioperera.github.io
```

Also works in the browser (mcpwasm-web) — the origin serves CORS.
